Exercise #1

Create a new folder called "learning_github", and add a couple of small text files to it from your computer.
Create a repository with this existing folder in GitKraken. Make sure to create it both locally, and remotely on github.com.
Create a new plain text file called "data-file.txt", add a line or 2 of content to it and save it to the "learning_github" folder.
Go to GitKraken, and commit the change with an approriate message.
Switch repos back to class repo.
